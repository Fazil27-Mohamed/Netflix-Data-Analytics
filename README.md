# Netflix Data Analytics 📊

## 📌 Project Overview

This project performs exploratory data analysis on the Netflix dataset containing 8,000+ records.
The objective is to identify patterns in content distribution, growth trends, and country-wise contributions.

---

## 🛠️ Tech Stack

* Python
* Pandas
* Matplotlib

---

## 📂 Dataset

The dataset includes information about Netflix movies and TV shows such as:

* Content type (Movie / TV Show)
* Title
* Country
* Release year
* Genre

---

## 🔍 Analysis Performed

* Compared distribution of Movies and TV Shows
* Analyzed year-wise growth of content
* Identified top countries producing content
* Examined most frequent genres

---

## 📊 Key Insights

* Movies are more prevalent than TV Shows on Netflix
* Significant growth in content production is observed after 2015
* USA and India are among the leading content producers
* Drama and International genres appear most frequently

---

## ▶️ How to Run

1. Download the dataset file (`netflix_titles.csv`)
2. Open the notebook (`netflix_analysis.ipynb`) using Jupyter Notebook or Google Colab
3. Run all cells to perform the analysis
